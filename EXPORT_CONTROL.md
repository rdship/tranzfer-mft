# Export Control Notice

## Classification

TranzFer MFT contains cryptographic functionality and is classified under:

- **ECCN**: 5D002 (Information Security Software)
- **CCATS**: Pending (self-classification under EAR Part 740.17)
- **Encryption**: AES-256, RSA, SHA-256, TLS 1.2/1.3, SSH (SFTP)

## U.S. Export Administration Regulations (EAR)

This software is subject to the U.S. Export Administration Regulations (EAR),
15 CFR Parts 730-774. This software is classified as ECCN 5D002, which covers
information security software using or performing cryptographic functions.

## License Exception TSU (EAR 740.13)

This software qualifies for License Exception TSU (Technology and Software
Unrestricted) under EAR 740.13(e), which permits export of publicly available
encryption source code, provided that:

1. The source code is publicly available (published on GitHub)
2. Email notification has been sent to BIS and NSA

## BIS/NSA Notification

As required by EAR 742.15(b) and 740.13(e), the following notification
has been submitted:

- **To**: Bureau of Industry and Security (BIS), U.S. Department of Commerce
  - Email: crypt-supp8@bis.doc.gov
- **To**: National Security Agency (NSA)
  - Email: enc@nsa.gov
- **Subject**: TSU Notification - TranzFer MFT
- **Content**: Publicly available encryption source code notification
  for TranzFer MFT, an enterprise managed file transfer platform
  using standard encryption (AES-256, RSA, TLS, SSH).
- **Repository**: https://github.com/rdship/tranzfer-mft

## Prohibited Destinations

This software may NOT be exported or re-exported to:

- Countries subject to U.S. comprehensive sanctions (currently: Cuba, Iran,
  North Korea, Syria, and the Crimea, Donetsk, and Luhansk regions)
- Any entity on the BIS Entity List, Denied Persons List, or
  OFAC Specially Designated Nationals (SDN) List
- Any end-use related to weapons of mass destruction

## User Responsibility

Users of this software are responsible for complying with all applicable
export control laws and regulations in their jurisdiction. By downloading
or using this software, you represent that:

1. You are not located in a sanctioned country or region
2. You are not on any U.S. government restricted party list
3. You will not re-export this software in violation of U.S. export controls

## ITAR Disclaimer

This software is NOT classified under the International Traffic in Arms
Regulations (ITAR). It contains only standard commercial encryption and
has no military or defense-specific functionality.
